Note
----
Beginning with the next version (0.9.0), JWST requires Python 3.5 and above.

Installing
----------

For developers:

    % git clone https://github.com/STScI-JWST/jwst.git

    % cd jwst

    % python setup.py install

    or

    % python setup.py develop
